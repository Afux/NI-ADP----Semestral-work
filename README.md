# ADP -- Semestral work
Tento repozitář obsahuje semestrální práci připravenou k obhajobě v předmětu NI-ADP.


## Tests
1) Testování funkčnosti mementa
2) Testovaní správného řetězení powerupů
3) Testování kolize

## **Features**
1) Přidání damage u rakety a hp u enemy. Vykreslení enemy se mění podle hp
2) Možnost nastavení předdefinovaných úrovní pomocí ***Builder***
3) Přidané power upy na rakety pomocí ***Decoratoru***
4) Přidaná kontrola před vykonáním příkazů v ***Proxy***
5) Přidání univerzálního systému kolizi pomocí ***Double Dispatch***
6) Přidaní ***Interpretera*** pro možnost ovládat hru pomocí příkazů
1) Vylepšení mementa pro ukládaní snapshotu do souboru s využitím circle bufferu
